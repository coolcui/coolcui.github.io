HTML特殊符号(HTML entity code)

2019-05-06

HTML特殊符号，也叫做HTML转义字符，实际上就是HTML实体码，即HTML entity code。HTML实体码没有任何附加功能、也不做任何改变，就是单纯的显示符号本身。

有两个空格符号需要区分一下， 和 

就是我们按下空格键后产生的空格，在HTML中显示时，浏览器会把多个空格压缩为一个空格： " "显示为" "

 或 全称是Non-breaking space，即不换行空格，表示不间断的空格，浏览器不会压缩，有多少个空格就显示多少个空格： "    "显示为" "
